# Grid Defense Overview

Built in IntelliJ, this is a Java Game where the user places down various towers to combat incoming light cycle and jet attacks. Each level increases difficulty but also yields more points to spend on 
more towers. The towers send out basic disk attacks, as well as an all-direction soundwave attack. 

The project utilizes a modular structure, as well as many interfaces which define Game Objects and Mechanics. The game utilizes a hand-drawn path that traverses the grid and 
provides a direction for targets to flow through. 

## How to run

In the terminal:

javac Checkpoint4/WelcomeToTheGrid.java

Then Run: 

java Checkpoint4.WelcomeToTheGrid
